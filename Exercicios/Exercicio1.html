Questão 1: Explique que vantagens e desvantagens existem no fato do JavaScript ser uma linguagem interpretada pelo navegador.

Vou dizer 3 vantagens que ele tem

A 1° vantagem seria a Interatividade onde o JavaScript permite que as paginas podem reagir a ações do usuário sem a necessidade de recarregar a página.


A 2° vantagem é a Compatibilidade: pois a maioria dos navegadores atuais o suporta ou seja, amaioria dos usuários pode executar o código JavaScript em suas páginas.


A 3° vantagem é a manipulação do DOM: pois ele permite que a manipulação da página da web. Isso significa que é possivel alterar o conteúdo, a aparência e a estrutura de um site após sua carga.

Desvantagens:

Umas das desvantagem é o Desempenho: Ele pode ser executado lentamente em comparação com outras linguagens de programação, o que pode afetar o desempenho da página.

Outra desvantagem é a Segunrança: como ele é executado no lado do cliente, o que significa que os usuários podem ver e modificar o código. Isso pode levar a vulnerabilidades de segurança.

Mais uma desvantagem é a Falta de tipagem estática : Como é uma linguagem de tipagem dinâmica, o que pode levar a erros em tempo de execução e dificultar a manutenção do código
